# EasyPostUI
WebApi接口测试工具 EasyPostUI.exe，是一个临时替代PostMan的WebApi调试小工具，绿色免安装单文件，支持Post和Get两种请求方式。

## 功能说明：
1、Post请求，自定义Header请求头信息，美化body-raw请求报文及响应报文。
2、Get请求，不限数量的参数
3、保存记录，点击发送按钮后，会将请求报文和响应报文记录在本地，方便下次打开工具时查看。
4、定时调用，支持指定间隔时长、多个并发次数的测试接口。
